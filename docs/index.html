CHasm ARM

Native AArch64 assembly tools for this Apple Silicon machine. Small sources, direct Darwin syscalls for file I/O, and local tests that prove the binaries run here.

Tools Build

Working Tools

This repo now contains real ARM64 assembly programs instead of only links to the upstream x86_64/Linux CHasm suite.

arm-echo

argv printer

Prints command arguments separated by spaces, then a newline.

arm-cat

file copier

Copies stdin or one or more files to stdout with a direct syscall loop.

arm-upper

byte filter

Uppercases ASCII bytes from stdin or files and leaves other bytes unchanged.

arm-lines

line counter

Counts newline bytes from stdin or files, with totals for multiple files.

Build

make all
make test
make install

Binaries are written to bin/. The build target is intentionally Darwin arm64 until another platform is tested.

Source

The assembly lives in src/, shared helpers live in src/common.S, and the local verification script is scripts/test.sh.
